2.vscode使用
    {快捷键：按 Ctrl+P，输入 "settings" 更改
    settings.json 更改全局设置
     "workbench.activityBar.visible": false, 
     //工具栏可见为true}

    vscode 设置分为三层，用户设置（也就是全局设置），工作区设置，
    和文件夹设置（我认为等同于项目设置），优先级由低到高

   插件 Auto Rename Tag     	同步修改标签
   插件 Live Server             浏览器随代码更新 ait+l+o打开 ait+l+c关闭
   插件 Code Runner         	可以直接在vscode里面运行调试代码
   插件 Search node_modules	ctrl+E搜索默认忽略node/modules，需要安装此插件 < node/modules来完成搜索文件

    npm: npm install --save-dev @types/node     让vscode由node.js的代码提示
